🚀 CodeCraft Pro
CodeCraft Pro is a modern, lightweight, and responsive online code editor built with React. It allows users to write and test HTML, CSS, and JavaScript code in real-time with an instant live preview.

🌟 Key Features
Live Preview: See your code changes rendered instantly in the browser.
Multi-File Support: Manage HTML, CSS, and JS files with an easy-to-use tabbed interface.
Persistent Storage: Your code is automatically saved locally, so you never lose your progress.
Theme Support: Toggle between multiple editor themes (VS Code Dark, Dracula, Monokai) for the best coding experience.
Interactive Console: Debug your JavaScript code using the built-in output console.


 #Tech Stack
Frontend: React (v19)

Editor: @monaco-editor/react (Professional coding interface)

Styling: CSS3 (Custom themes & Responsive layout)
